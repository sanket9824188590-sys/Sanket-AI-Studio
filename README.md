# 🚀 Sanket AI Studio

Free and Open Source AI Talking Video Studio

---

## 🎯 About

Sanket AI Studio is an open-source project for creating realistic AI talking videos from an image and audio.

The goal is to build one simple platform that supports multiple AI talking video models.

---

## ✨ Features

- 📷 Image Upload
- 🎤 Audio Upload
- 🎬 AI Talking Video Generation
- 🖥️ Google Colab Support
- 💻 Windows Support (Future)
- 🆓 Free and Open Source

---

## 🤖 Supported Models (Planned)

- InfiniteTalk
- MuseTalk
- LivePortrait
- SadTalker
- Wav2Lip
